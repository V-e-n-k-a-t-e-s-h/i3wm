To install i3wm

### Debian based distro
sudo apt-get install i3-gaps rofi dmenu i3locks i3blocks

### fedora
sudo dnf install i3-gaps rofi dmenu i3lock i3blocks

### Arch
sudo pacman -S i3-gaps xorg xorg-xdm dmenu i3status i3lock i3blocks
